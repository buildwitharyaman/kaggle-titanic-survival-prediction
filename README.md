# 🚢 Titanic Survival Prediction

An end-to-end machine learning project for the Kaggle Titanic competition. This project explores the complete ML workflow—from data exploration and preprocessing to feature engineering, model training, evaluation, and Kaggle submission generation.

## 📌 Project Overview

The objective is to predict whether a passenger survived the Titanic disaster based on demographic and travel information such as age, sex, ticket class, fare, and family size.

This project focuses on building a complete machine learning pipeline while following good data science practices rather than simply achieving the highest leaderboard score.

## 🎯 Objectives

- Perform comprehensive Exploratory Data Analysis (EDA)
- Handle missing values and clean the dataset
- Engineer meaningful features
- Train and compare multiple classification models
- Select and optimize the best-performing model
- Generate predictions for Kaggle submission

## 📂 Dataset

The dataset is provided by the Kaggle Titanic competition and contains passenger information including:

- Passenger demographics
- Ticket details
- Cabin information
- Family relationships
- Survival status (training data)

## ⚙️ Machine Learning Workflow

### 1. Exploratory Data Analysis (EDA)

- Dataset inspection
- Missing value analysis
- Feature distributions
- Correlation analysis
- Survival analysis across different features

### 2. Data Preprocessing

- Handling missing values
- Encoding categorical variables
- Removing unnecessary columns
- Preparing datasets for modeling

### 3. Feature Engineering

- Created new features from existing information
- Improved feature representation
- Prepared numerical inputs for machine learning models

### 4. Model Building

Multiple classification algorithms were trained and evaluated to identify the best-performing model.

### 5. Model Evaluation

Models were compared using validation accuracy before selecting the final model.

### 6. Kaggle Submission

The final model was used to generate predictions for the competition test dataset.
